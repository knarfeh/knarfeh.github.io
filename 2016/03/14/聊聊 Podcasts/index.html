聊聊Podcasts

发表于 2016-03-14   |  

大一那年暑假，骑行拉萨前，买了160G，能存两万首歌的IPod classic。

我在里面塞了各种类型的Podcasts、有声书，甚至还下载了5季生活大爆炸转换了格式塞里面，走的时候还怕自己不乐意听，结果呢，2160公里的旅途，20来天，几乎每天戴着耳机超过10小时。听了几本想要重看的书：《兄弟》，《小王子》，《苏菲的世界》，《民主的细节》，听各种Podcasts，包括：潘吉Jenny告诉你，ESL，听说很动听，圣人请卸妆等等。从那时候开始听Podcasts。

后来的两年时间，由于室友都是不同专业的，我和室友们总是不同步，吃饭、走在路上，我总是一个人，这段时间里仍然是Podcasts陪着我，后来慢慢开始听Talk Python To Me，程序员之路这些，感觉就像是发现了一个更大的世界，听着听着就对几位主播很感激，一方面是因为他们帮我开阔了眼界，另一方面也因为他们陪着我度过那些岁月，竟变得有些依恋Podcasts了，慢慢地变成了一个Podcasts重度用户。

我热爱音乐，后来听音乐少了，几乎带上耳机就听Podcasts，而Podcasts，很多如主播不鳥萬如一所说的那样，如湿货一般，是「无用的、主观的、热烈的、有力的、情感的」「音頻廣播是一種很特別的媒介，比視頻、圖畫和文字都更能直接訴諸情感」，它陪着我，让我感觉到「和主播身处一个共同空间」。以前也有过听收音机的经历。读初中时，我只身到YY读书，也听过一段时间，后来和同桌熟络了，交了几个好朋友，也就没有再听过。所以，听「电台」对于我来说是一种奇妙的体验，一直想写篇文章聊一聊，一直憋着拖着，终于今天动笔了。

为什么要听Podcast

开拓眼界

不说聊聊比特币背后的技术和Blockchain，Talk Python To Me 这样的打开我新世界的节目，说个小事情，听内核恐慌，吴涛谈到将Caps Lock键映射成Control按键会轻松很多，我茅塞顿开，尝试了一下，发现真的给自己带来了很多便利，转而就想怪自己：我怎么之前没想到？以前一直觉得用小拇指按Control键太别扭，现在敲键盘真是太舒服了，而且Caps Lock键用做大小写转换的使用率几乎为零，用shift就够了，映射按键后也提高了按键的使用率。这些东西书里可学不到，博客里也很少有人写，但这些生活经验是需要交流的，这样才能跳出来发现新的舒适区。Podcast用声音来传递信息，尽管相比于文字的方式比较低效，但这种方式更像一种生活中的交流，感性，让人沉迷，在不经意间就可以学到很多。

提升技能

想要学习某语言，听Podcasts可以很有帮助。你可以听各种语言教程，拿英语来说，有ESL Podcast，Daily English Expression Podcat，The English We Speak 等等，恶魔的奶爸非常推崇ESL，我也持同样观点，ESL真的很可贵，1000多集都是免费的，话题涵盖范围也非常广，很有时效性，这个Podcast必然会留存于历史的，很多年后也还会有价值，要靠Podcast学英语，听这一个就够了。不仅仅是语言的学习，你还可以通过听这些了解到语言背后的人文，比如你可以听Stuff You Should Know，The China History这些，既可以学英语，又可以满足自己的精神需求。

想提高专业技能？我相信你可以找到你需要的关于专业知识的Podcast，就我来说，我最近在听这几个技术播客：Talk Python To Me，Taking Machines，内核恐慌，Teahour.fm，IT公论，当然，这些不会做到手把手教写代码，真要说提升编程能力，自然还是整天看书写代码读博客写文章来的实在，但你能在路上、吃饭的时候听这些Podcast，难道还能要求更多吗？

娱乐

Podcasts 里面的资源真的很多，有成千上万的人在把声音留存于这世界，其中自然有不少逗逼，纯粹求一乐，不爱听某人一本正经之余的幽默，可以听脱口秀，可以听相声，如赵畅不误，三主持人各种闲聊，挺逗；圣人请卸妆，朗读咪蒙（就是写《好疼的金圣叹》的那位）写的文章，以现代人的角度调侃式的方式记录历史人物，非常有趣（话说这节目会不会有版权问题？）。

在Podcasts 里面，你几乎可以找到一切你感兴趣的娱乐内容，喜欢电影你可以听电影不无聊，在这你可以听到电影的最新资讯，各种电影专题，好片烂片评论，可以听VoiceClub 电台(改名了，以前叫「VC 电影说？」)，时长大多10min左右，节目大多是影片短评，新片预告，也挺有意思，还有没节操有情操电影影评，英语的纯影评，特点：没节操，有情操，还有Now Playing Presents: The Avengers Movie Retrospective Series，聊复仇者联盟；喜欢音乐你可以听摇滚天堂，lastradio音乐广播，ROAR音乐电台。摇滚天堂会聊一些小众音乐，lastradio音乐电台的主持人是东北银（邀请过同是东北的–后摇乐队惘闻的–键盘手，刀哥），听起来主持人也搞过乐队，有一集谈到汪峰，特鄙视汪峰，聊到了汪峰的历史，挺有观点，挺有意思，ROAR音乐电台貌似是一大学生弄的，主持人声线一般，聊几句，放几首歌，有时候歌都放完了，主持人还没回来，然后节目也没剪，直接放Podcasts让人下载了，不过推荐的音乐确实不错，电音，后摇的两期就特别好。娱乐嘛，无非舒畅身心打发时间，而这些质量都非常高，完全免费，几乎没广告，还要啥自行车啊:)

满足兴趣

我一直对v2ex的站长Livid很感兴趣，没想到有一天翻到了Livid的访谈，听了之后才了解到v2ex这个网站的初衷以及很多技术细节，能够这么「近距离」听到「偶像」的声音的心情就像是捡到宝贝一样。在Mac上写markdown，mou是我第一个md编辑器，后来了解到mou持续跳票，很多人指责罗晨，twitter上已经有各种投诉，知乎上也有相关讨论，听了养花、养草、种人参、玩热解石墨、写 Mac 软件这一集可以真真切切「接触」这个人。Podcasts是一个世界广场，和微博不一样，这里存在的是人们更真实的声音，在这广场穿梭，指不定什么时候就听到了自己感兴趣的内容，这是非常难得的，放在一百年前，大概是想都不敢想的事情。

总之呢，与其在路上中盯着脚下或望着千篇一律的风景，吃饭的时候听电视里强塞给你的各种国家大事，不如就听听Podcast，关注一下内心吧。

听什么Podcast？

我在上面已经提到了一些，都是我听过的，有一些简单的评价，有兴趣的可以自己去找找。几年前我下载的Podcast总数是3000+，我当时想，每天听一个小时左右，一年怎么也可以听500吧，然而我现在下载的Podcast总数快4000…

这时代的信息是爆炸式的，相比之下我听的Podcast实在太少太少了，汲取的营养也太少了，贡献的力量几乎是零，实在无法给出非常宏观的推荐，我觉得还是按照自己的兴趣去搜索比较好，所以我就不给出我听过的Podcast列表了。这里也提出一个倡议，希望听Podcasts的朋友多对听过的节目做评价，不打字，打个星点个赞也是好的，让优秀的浮出水面，差劲的销声匿迹，参与便是功德了，节约的是大家的时间，就算播客多是感性的，也还是有好有差，听差劲的内容，还不如不听。所以不要闷声听，我们也要参与进来，不管写点什么，写多少，只要是建设性的，都是我们为这社会更美好添加的一砖一瓦，「写给别人看，你就成为更广阔世界的一部分，发表文字就是在宣扬自己是社会一员，愿意做一些有意义的贡献」。多说一句，看电影，看书都是如此，这也是豆瓣、知乎这类网站如此有益的原因。

以前我也喜欢闷声听，尽管听的时候也在思考，在刻意记忆，但我并没有把这些输出，其实这是不符合客观规律的，有大量的输入，也要有输出才行，所以我打算把我听的Podcast做个记录，并且把这些选择性地放在博客上，有的可能是一个简单的评价或记录，有的可能多写一点，刚开始记录，会不定时更新，慢慢来吧。我相信这些是有帮助的，像Talk Python To Me，Teahour.fm这些，几乎都是干货，听完就忘可不好，记下来，于我这是一种整理，于他人，也可以让刚开始听的朋友多一点了解，也多一点选择。再具体点，比如我把 Talk Python To Me文稿记下来，甚至翻译，对于我是一种提高，对于英语不太好的朋友也有帮助（说起文稿，突然想起来，如果teahour.fm将所有节目内容整理成文字出一本书，尽管我还是学生，但是，就算这书1000块一本我也是愿意买的，不知道有多少人也有这个需求呢？）。

有的播客确实是干货满满，花时间背下来都值得，如ESL Podcast，每一集都可以在官网找到文稿，甚至还可以买对应的书学习，再比如我喜欢的 Talk Python To Me ，主持人采访世界各地的Python开发者，讲顶级开发者的故事，介绍各种库、开发工具的前世今生，把这些内容内化于心，可以学英语，也是为了和国际友人瞎BB:）于人于己都是好事，何乐而不为呢？

总之，这时代允许我们将世界变成一个广场，让每个人的闪光点都能发挥出来，那我们就好好享受吧。
